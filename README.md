# 📊 Product Process Management System

## 🚀 Overview
The **Product Process Management System** is a data-driven application designed to manage, track, and analyze the
complete lifecycle of product-related data. It allows users to import data, update workflows, monitor status with
timestamps, and visualize insights through dashboards.

---

## ✨ Features

### 📥 1. Data Import
- Upload CSV or Excel files
- Automatically read and display structured product data
- Supports multiple attributes like Order Number, SKU, Dates, Jewelry details, etc.

---

### ✏️ 2. Data Management
- Add new columns dynamically
- Delete unwanted columns
- Modify and update existing data
- Flexible workflow customization

---

### 🔄 3. Process Tracking
- Track product status (e.g., Pre-Order, Active, Out of Stock)
- Automatic timestamp recording for status updates
- Monitor product lifecycle stages

---

### 📊 4. Dashboard & Visualization
- Bar charts for status distribution
- Pie charts for overall insights
- Easy-to-understand visual analytics
- Helps in decision-making

---

### 📤 5. Export Functionality
- Export processed data
- Useful for reporting and sharing

---

## ⚙️ Tech Stack

- **Frontend:** Streamlit   
- **Data Handling:** Pandas  
- **Visualization:** Plotly  
- **File Handling:** OpenPyXL  

---

## 🧠 How It Works

Upload → View → Modify → Update Status → Track Time → Visualize → Export
